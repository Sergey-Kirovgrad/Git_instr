

Что такое Git и зачем он нужен?

Git - это консольная утилита, для отслеживания и ведения истории
изменения файлов, в вашем проекте. Чаще всего его используют
для кода, но можно и для других файлов.

Проект был создан Линусом Торвальдсом для управления разработкой ядра Linux, первая версия выпущена 7 апреля 2005 года. На сегодняшний день его поддерживает Джунио Хаман.
